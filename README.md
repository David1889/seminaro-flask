## 🚀 Instrucciones de uso

1. Clonar este repositorio y entrar en la carpeta del proyecto.  
2. Instalar dependencias: pip install flask
3. Ejecutar el servidor: flask run
4. Abrir el navegador: Ir a http://127.0.0.1:5000
